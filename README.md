# My Cafe - Simple Flask Project

A minimal Flask app with a homepage and a menu page.

Quickstart (Windows, bash):

```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
python app.py
# open http://127.0.0.1:5000 in your browser
```

Run tests:

```bash
pip install -r requirements.txt
pytest -q
```
